Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

This project is to visually display data that was collected from people that volunteered to give their bellybutton bacteria for a study to show different bacteria cultures in each person's bellybutton. Each graph has its own purpose and is useful to give people insight into what cultures exist in their bullybuttoms. Researchers are using this data to help determine ways to use the bacteria cultures that are suitable for synthetic beef production.
